# Configure DevPulse

Key settings you can customize:

| Setting | Default | Description |
|---|---|---|
| Idle Threshold | 5 min | When to stop counting active time |
| Focus Goal | 90 min | Default focus session duration |
| Status Bar Format | time+intent | What shows in the status bar |
| Data Retention | 90 days | How long to keep historical data |
| Exclude Patterns | node_modules, .git | Files to ignore |

All settings are under `devpulse.*` in VS Code settings.
