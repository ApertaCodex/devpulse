# The DevPulse Dashboard

The dashboard gives you a rich visual overview of your productivity:

- **Today Tab** \u2014 Hourly heatmap, work intent breakdown, top languages
- **Weekly Tab** \u2014 7-day trend, top projects, context switch rate
- **AI Insights Tab** \u2014 Productivity score, coaching tips, peak hours
- **Focus Tab** \u2014 Active session controls, recent session history

Open it anytime with `Ctrl+Alt+D` / `Cmd+Alt+D`.
