# The DevPulse Dashboard

The dashboard gives you a rich visual overview of your productivity:

- **Today Tab** — Hourly heatmap, work intent breakdown, top languages
- **Weekly Tab** — 7-day trend, top projects, context switch rate
- **AI Insights Tab** — Productivity score, coaching tips, peak hours
- **Focus Tab** — Active session controls, recent session history

Open it anytime with `Ctrl+Alt+D` / `Cmd+Alt+D`.
